  2. 编程基础（2）——函数

    在数学中，一个函数长这样 f(x) = x + 1。f 看不出什么意义，为了让它有意义，可以把函数名变长，想 add_one(x) = x + 1。 同理，这个 x 也没有什么意义，也变长一点，可以变成 add_one(number) = number + 1。

    在这里，形同 add_one number 的这些名字被称之为标识符，标识符一般可以由英文字母（注意，几乎所有语言都是区分大小写的）， 下划线（即 _），和部分 Unicode 字符（如汉字~~，ruby 甚至可以用不间断空格（一个看起来像空格的字符）~~）组成。

    既然标识符一般可以用汉字，当然可以写出 加一(数字) = 数字 + 1 …
