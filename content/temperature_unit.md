为什么不在温度单位后加常数表示实际温度

如用地球的温度单位
```
(20°C - 10°C) = 2(15°C - 10°C)
```
而 `10°C` 却不等于 `2 * 5°C`

加上常数后
```
((20°C + Tc) - (10°C + Tc)) = 2((15°C + Tc) - (10°C + Tc))
```
`10°C + Tc` 自然不等于 `2(5°C + Tc)`

单位换算也很清楚了
```
1K = 1°C
Tc = Tk + 273.15°C
1°C = 1.8°F
Tf = Tc + 32°F
```
